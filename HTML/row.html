Row

The Row Widget in Flutter In Flutter, the Row widget is a fundamental layout widget that arranges its children horizontally. It's a versatile tool for creating horizontal layouts, allowing you to align and space multiple widgets side-by-side.   Key Properties of a Row:

  • children: A list of widgets to be displayed horizontally.  
    • mainAxisAlignment: Controls how the children are aligned along the main axis (horizontally). Common values include:
  • MainAxisAlignment.start: Aligns children to the start.
  • MainAxisAlignment.end: Aligns children to the end.
  • MainAxisAlignment.center: Centers the children.
  • MainAxisAlignment.spaceAround: Spaces children evenly, with half-size spaces at the start and end.
  • MainAxisAlignment.spaceBetween: Spaces children evenly, with no space at the start and end.
  • MainAxisAlignment.spaceEvenly: Spaces children evenly, with equal spaces between and around the children.
    •   crossAxisAlignment: Controls how the children are aligned along the cross axis (vertically). Common values include:
  • CrossAxisAlignment.start: Aligns children to the start.
  • CrossAxisAlignment.end: Aligns children to the end.
  • CrossAxisAlignment.center: Centers the children.
  • CrossAxisAlignment.stretch: Stretches the children to fill the available space.
    •  
  • mainAxisSize: Controls how the Row widget sizes itself. Common values include:
  • MainAxisSize.max: The Row will expand to fill the available space.
  • MainAxisSize.min: The Row will be as small as possible to fit its children.

                        
                            Row(
   mainAxisAlignment: MainAxisAlignment.spaceEvenly,
   crossAxisAlignment: CrossAxisAlignment.center,
  children: [
   Icon(Icons.home),
    Text('Home'),
  Icon(Icons.search),
     Text('Search'),
  Icon(Icons.settings),
     Text('Settings'),
   ],
      )
