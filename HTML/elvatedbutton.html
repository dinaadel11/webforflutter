ElevatedButton

ElevatedButton in Flutter In Flutter, ElevatedButton is a material design button that appears elevated off the surface of the screen. It's commonly used for primary actions and is visually distinct from other button types. Key Properties:

  • onPressed: A callback function that's triggered when the button is pressed.  
  • child: The widget displayed inside the button, usually a Text widget.
  • style: A ButtonStyle object that allows you to customize the button's appearance, including:
  • backgroundColor
  • foregroundColor
  • elevation
  • shape
  • textStyle
  • And more
    • Key Points:
  • Elevation: The button appears raised above the surface.  
  • Style Customization: You can tailor the button's appearance to match your app's theme.  
  • Accessibility: Ensure the button is accessible to users with disabilities by providing appropriate semantic information.
  • Responsiveness: Consider how the button will look and behave on different screen sizes.
    • By effectively using ElevatedButton, you can create visually appealing and interactive user interfaces in your Flutter apps. 

                        
      ElevatedButton(
      onPressed: () {
      // Handle button press
              },
      style: ElevatedButton.styleFrom(
      primary: Colors.blue,
      onPrimary: Colors.white,
      shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(10.0),
           ),
           ),
      child: Text('Custom Elevated Button'),
           )
