Text

In Flutter, the Text widget is a fundamental component used to display a string of text with a specific style. Here's a brief overview:

  • Purpose: The Text widget displays a run of text and can be styled in various ways.
  • Single Style: It presents the text with a single style, which is determined either by properties set directly on the widget or inherited from the surrounding DefaultTextStyle.
  • Layout Flexibility: Depending on the layout constraints, the text can either be displayed on a single line or can break across multiple lines.
  • Usage: It's one of the most commonly used widgets for displaying text in a Flutter app.
You can customize the text appearance using properties such as style, allowing you to define fonts, colors, sizes, and more.

                            
           
     Text('Hello, Flutter!'),
     child: Text(
         'ليس لدي حساب بالفعل ',
     style: TextStyle(color: Colors.white),
                ),

         'تسجيل',
     style: TextStyle(color: Colors.white),
