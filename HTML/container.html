Container

The Container Widget in Flutter In Flutter, the Container widget is a versatile building block that combines several common painting, positioning, and sizing properties into a single widget. Think of it as a customizable box that you can use to style and position your UI elements. Key Properties of a Container:

  1. width and height: These properties explicitly set the dimensions of the container.  
  2. color: Defines the background color of the container.  
  3. padding: Adds space between the container's edges and its child content.  
  4. margin: Creates space around the entire container.  
  5. decoration: Allows you to apply more complex decorations, such as borders, shadows, gradients, and more.  
  6. alignment: Positions the child content within the container.  
  7. transform: Applies transformations like rotation, scaling, and skewing to the container.

                        
                            Container(
   width: 200,
     height: 100,
     color: Colors.blue,
     padding: EdgeInsets.all(16),
     margin: EdgeInsets.all(8),
     decoration: BoxDecoration(
      borderRadius: BorderRadius.circular(10),
    border: Border.all(color: Colors.black),
         ),
  child: Text(
       'CONTIRAN',
   style: TextStyle(color: Colors.white),
          ),
           )
